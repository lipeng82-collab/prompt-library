# 湘小配 · 全资产配置管家 使用说明

> 版本：v1.0 | 创建：2026-04-02 | 湘江研究院研究中心

---

## 一、功能概述

**湘小配**是湘江研究院研究中心的全品类资产配置管家Agent，统筹管理不动产、有价证券、贵金属、期货、数字货币等所有投资属性资产，从**全局视角**提供配置分析、风险评估和动态再平衡建议。

### 与现有Agent的关系

```
                    ┌─────────────────┐
                    │    湘小配        │  ← 全局配置"总指挥"
                    │  全资产配置管家   │
                    └────────┬────────┘
                             │ 统筹调度
              ┌──────────────┼──────────────┐
              ▼              ▼              ▼
     ┌────────────┐  ┌────────────┐  ┌────────────┐
     │  湘小A系列  │  │   湘小金    │  │   湘小币    │
     │ A股/基金   │  │ 黄金/大宗  │  │ 数字货币   │
     │ 固收/选股  │  │ 美元/原油  │  │ 行情/策略  │
     └────────────┘  └────────────┘  └────────────┘
```

**核心差异**：湘小A/湘小金/湘小币回答的是"这个品类该不该买"，湘小配回答的是"你的整体财富配置健不健康"。

## 二、支持的资产品类

| # | 大类 | 细分 | 数据源 |
|---|------|------|--------|
| 1 | 不动产 | 住宅/商业/土地/车位 | 用户录入+公开评估 |
| 2 | 有价证券 | A股/港股/美股 | AkShare（湘小A系列） |
| 3 | 基金 | ETF/偏股/偏债/货基 | AkShare |
| 4 | 固收/债券 | 国债/信用债/存单 | 用户录入+AkShare |
| 5 | 贵金属 | 黄金/白银 | 湘小金Agent |
| 6 | 期货 | 商品期货/金融期货 | AkShare |
| 7 | 数字货币 | BTC/ETH/主流币 | 湘小币Agent |
| 8 | 现金等价物 | 活期/货基/理财 | 用户录入 |

### 菜单功能

```

7  编辑资产组合       → 打开JSON配置文件
0  退出
```

## 四、资产录入

### 方式一：编辑JSON配置文件

首次运行会自动生成 `portfolio.json`，包含示例数据。用文本编辑器打开，替换为你的真实资产：

```json
{
  "owner": "你的姓名",
  "currency": "CNY",
  "risk_profile": "moderate",
  "assets": {
    "real_estate": [
      {
        "name": "深圳市南山区XX小区",
        "type": "住宅",
        "purchase_price": 3500000,
        "current_value": 4200000,
        "mortgage": 1500000,
        "liquidity": "low"
      }
    ],
    "equities_a": [
      {
        "name": "贵州茅台",
        "code": "600519.SH",
        "shares": 10,
        "cost_basis": 160000,
        "current_price": "auto"
      }
    ],
    "crypto": [
      {
        "name": "Bitcoin",
        "symbol": "BTC",
        "quantity": 0.5,
        "avg_cost": 480000,
        "current_price": "auto"
      }
    ]
  }
}
```

### 方式二：在对话中录入

直接告诉我你的资产清单，我会帮你生成JSON配置文件。示例：

> "帮我录入：深圳1套房估值420万贷款150万、茅台10手成本16万、招行1000股成本3.5万、沪深300ETF 5000份成本1.8万、黄金200克成本480/克、BTC 0.5个成本48万、大额存单20万、活期5万+余额宝10万"

## 五、报告模块说明

### 完整配置报告（7大模块）

| 模块 | 内容 | 关键指标 |
|------|------|---------|
| 一、资产全景 | 各品类市值/占比 | 总资产、品类数 |
| 二、配置结构 | 实际vs目标配置 | 偏离度 |
| 三、风险诊断 | 集中度/VaR/流动性 | HHI、分散化比率、VaR(95%/99%) |
| 四、压力测试 | 6大极端场景 | 预估损失、评级 |
| 五、再平衡建议 | 调整方案 | 操作方向、优先级、调整额 |
| 六、收益预期 | 组合预期年化 | 加权预期收益率 |
| 七、相关性矩阵 | 品类间相关系数 | 热力图数据 |

### 六大压力测试场景

| 场景 | 核心冲击 |
|------|---------|
| 2008全球金融危机 | 股-40%/房-20% |
| 2020新冠疫情冲击 | 股-30%/币-50%/金+15% |
| 2022加息周期 | 股-20%/债-15%/币-65% |
| 人民币贬值10% | 外币资产+人民币资产- |
| 房产市场下行20% | 房-20%/股-15% |
| 币圈极端崩盘 | 币-80% |

## 六、三种预设配置方案

### 保守型（风险厌恶）
不动产30% + 固收30% + 现金15% + 黄金10% + 证券10% + 币0%

### 均衡型（风险中性）⭐ 默认
不动产30% + 证券25% + 固收15% + 黄金10% + 现金10% + 币5% + 其他5%

### 进取型（风险偏好）
证券30% + 不动产25% + 币10% + 固收12% + 黄金8% + 期货5% + 现金5% + 其他5%

## 九、风险提示

> **重要声明：本工具仅供内部研究参考，不构成任何投资建议。**

1. 所有分析基于历史数据和统计模型，不预测未来
2. 相关性矩阵和波动率数据为市场经验值，实际情况可能偏离
3. 不动产估值需要专业评估，自行录入可能不准确
4. 数字货币价格波动极大，仓位需严格控制
5. 投资有风险，决策需谨慎
