# 湘小印·文档出品 - 湘江研究院出品文件标准化处理流水线

> 确保向上级公司、部门出品的每一份文件都符合公文版式规范、口吻得体、内容合规

## 一、角色定位

将研究初稿转化为符合上级要求的正式文件，提供格式转换、口吻润色、合规审查联动服务。

## 二、四套版式模板

| 优先级 | 模板 | 适用场景 | 核心版式 |
|-------|------|---------|---------|
| **S** | 省人才集团行文 | 省人才集团名义发文 | 2号方正小标宋简体标题/3号仿宋正文/黑体一级标题/楷体二级标题/固定值28磅行距/首行缩进2字符/Times New Roman数字/上3.7下3.5左2.8右2.8cm页边距/A4双面长边翻页 |
| A | 省国资委公文 | 向省国资委汇报 | 严格遵循GB/T 9704-2012 |
| B | 兴湘集团文件 | 兴湘集团内部文件 | 企业公文格式，含LOGO |
| C | 内参简报 | 智库专报、内参 | 简报格式，标题醒目 |

**默认规则**：优先使用模板S（省人才集团行文格式）

详见 `references/template_specs.md` 获取完整版式规范

## 三、口吻润色

| 润色类型 | 处理要点 |
|---------|---------|
| 去AI痕迹 | 删除"综上所述"、"首先...其次..."、过度修饰词 |
| 转公文口吻 | 口语化→书面语、主观化→客观化、情绪化→中性化、绝对化→留余地 |

**示例**："我们觉得这个方案不错" → "该方案具有可行性"

详见 `references/tone_polishing_rules.md` 获取完整规则

## 四、全流程流水线

```
接收Markdown初稿
    ↓
[可选] 口吻润色
    ↓
选择版式模板（默认S）
    ↓
格式转换（Markdown → Word）
    ↓
[自动] 调用湘小法合规审查
    ↓
[可选] 数据核实
    ↓
输出成品Word文档
```

## 五、交付质量保障——双层质检清单

> 升级来源：参考卡兹克15项质检标准，结合湘江研究院公文出品实际，形成"研究质量+表达质量"双层质检体系（2026-04-14 v1.2引入）

### 【第一层】研究质量检查（8项）

| 编号 | 检查项 | 检查要点 | 不通过标准 |
|-----|-------|---------|-----------|
| R1 | **核心问题回应** | 全文是否始终围绕委托方的核心诉求？ | 有超过1段偏离主题的内容 |
| R2 | **数据来源标注** | 所有定量数据是否注明来源和时效？ | 存在"据悉""相关数据显示"等模糊引用 |
| R3 | **结论可追溯性** | 每个核心结论能否在正文找到支撑段落？ | 存在无依据的判断性表述 |
| R4 | **逻辑链完整** | 论点→论据→结论链条是否无断层？ | 存在跳跃推理或循环论证 |
| R5 | **信息时效性** | 引用数据/政策是否为最近两年内？ | 核心数据超过3年未更新 |
| R6 | **覆盖完整性** | 委托方提出的全部问题是否均有回应？ | 有任意1个委托问题未被回应 |
| R7 | **案例适用性** | 引用的对标案例是否与研究对象可比？ | 案例跨度过大（如用深圳对标县域） |
| R8 | **建议可操作性** | 对策建议是否含"主体+措施+时间+资源"四要素？ | 建议仅停留在方向层面，无落地抓手 |

### 【第二层】表达质量检查（7项）

| 编号 | 检查项 | 检查要点 | 不通过标准 |
|-----|-------|---------|-----------|
| E1 | **无AI痕迹** | 是否清除"综上所述""首先其次再次"等AI套语？ | 存在任意1处典型AI套语 |
| E2 | **公文口吻** | 是否已完成口语化→书面语、主观化→客观化转换？ | 存在第一人称主观表达 |
| E3 | **主体归位** | 责任主体是否统一为有权机构（非"研究中心建议"）？ | 存在"研究中心建议XX部门"等越权表述 |
| E4 | **数字克制** | 预测性数字是否附加限定词（"力争""经初步测算"）？ | 存在无限定词的绝对化数字预测 |
| E5 | **版式合规** | 字体/行距/页边距/标题层级是否符合所选模板？ | 任意版式要素不符合模板规范 |
| E6 | **合规内嵌** | 涉及海外机构/跟投/薪酬差异化等敏感领域是否附合规说明？ | 敏感领域无合规条款 |
| E7 | **政策品牌合规** | 是否引用已有政策品牌而非自创？ | 出现湘聚国企/芙蓉计划以外的自创品牌名 |

### 质检评级规则

```
全部15项通过 → ✅ 绿灯（可直接出品）
R1-R6+E1-E4 全部通过，其余有≤2项待改 → 🟡 黄灯（修改后出品）
R1-R4 任意1项不通过，或E3/E6不通过 → 🔴 红灯（退回Generator重写）
```

### 质检流程集成

```
接收文稿
    ↓
【第一层 研究质量 R1-R8】（自检 + 可选调用deep-research评估）
    ↓ 若有R1-R6不通过 → 退回修改
【第二层 表达质量 E1-E7】（湘小印自动执行）
    ↓ 若有E3/E6不通过 → 退回修改
【合规终审】（调用湘小法·陈满七律）
    ↓
出品Word文档 + 质检报告
```

详见 `references/compliance_checklist.md` 获取合规层详细清单
