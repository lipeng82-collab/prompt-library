# 湘小法·输出格式模板

> 本文件为湘小法SKILL.md的参考附件，定义法律意见书、风险提示函、合规确认函的标准输出格式。

---

## 一、法律意见书（标准格式）

```markdown
# 法律意见书

**编号**：[年份]-[序号]  
**日期**：2026年X月X日  
**审核事项**：[事项名称]  
**审核类型**：[风险识别/合规审查/投资尽调]  

## 一、审核事项概述

[简要描述需要审核的内容]

## 二、风险识别

| 风险类别 | 风险描述 | 风险等级 | 建议措施 |
|----------|----------|----------|----------|
|          |          |          |          |

## 三、法律分析

[依据法律法规进行分析]

## 四、结论与建议

### 4.1 结论
[是否可以通过 / 需要补充材料 / 建议拒绝]

### 4.2 建议措施
[具体整改建议]

---
**审核人**：湘小法·国资护法  
**复核**：（可选人工复核）
```

## 二、风险提示函

```markdown
# 风险提示函

**致**：XX部门/项目组  
**事项**：XX项目风险提示  

## 风险事项

[描述发现的风险点]

## 风险分析

[分析风险可能造成的影响]

## 整改建议

1. [建议一]
2. [建议二]

**提示**：请在X个工作日内反馈处理结果
```

## 三、合规确认函

```markdown
# 合规确认函

**编号**：[年份]-[序号]  
**确认事项**：[事项名称]  
**确认结果**：✅ 已通过合规审核  

## 审核要点

- [审核要点1] ✅
- [审核要点2] ✅

## 有效期

本确认函有效期为X个月，届时需重新审核

---
**确认人**：湘小法·国资护法
```
