# 湘江研究院 AI Agent 工具箱 v1.0

## 工具箱简介

本工具箱整合了湘江研究院 Agent 体系全部核心能力，共 23 个专业化 Skill，覆盖政策研究、专报撰写、产业分析、数据处理、宏观经济研判、学术写作等全流程智库业务场景，同事安装后可在 WorkBuddy 中直接调用使用。

---

## 安装说明

### 方式一：一键安装（推荐）

1. 解压工具箱压缩包
2. 双击运行 `install.bat`
3. 重启 WorkBuddy，完成

### 方式二：手动复制

将 `skills/` 目录下的各个文件夹复制到：

```
C:\Users\[你的用户名]\.workbuddy\skills\
```

## 典型使用场景

### 场景 1：撰写国资国企改革专报

```
用户：使用湘小策，帮我撰写一篇关于湖南省属国企战略性新兴产业布局的专报
湘小策 → 调用 policy-tracker-analyst（政策梳理）
       → 调用 industry-chain-analysis（产业分析）
       → 调用 data-analysis-xt（数据支撑）
       → 调用 china-macro-analyst（宏观研判）
       → 输出专报草稿
湘小印 → 标准化审核与格式优化
```

### 场景 2：分析城投退平台转型路径

```
用户：使用湘小研，分析某地市城投退平台的转型路径
湘小研 → 调度 deep-research（深度研究）
       → 调用 industry-chain-analysis（产业链机会）
       → 调用 data-analysis-xt（数据量化）
       → 综合输出转型分析报告
```

### 场景 3：申报省国资委课题

```
用户：使用湘小协，帮我申报省国资委2026年重点课题
湘小协 → 调用 proposal-writing-expert（申报书撰写）
       → 调用 deep-research（背景研究）
       → 调用 mckinsey-mece（MECE框架）
       → 输出完整课题申报书
湘小印 → 出品审核
```

### 场景 4：季度研究报告编写

```
用户：使用湘小协，完成本季度省属国企改革进展报告
湘小协 → 调用 policy-tracker-analyst（政策动态）
       → 调用 china-macro-analyst（宏观环境）
       → 调用 data-analysis-xt（数据更新）
       → 输出季度报告
湘小印 → 格式标准化
```
