三、典型使用场景

1 国资国企改革专报撰写
湘小策 接收任务 → 政策追踪 政策梳理 → 产业分析 格局分析 → 数据分析 量化支撑 → 宏观分析 环境研判 → 专报撰写 成稿输出 → 湘小印 审核出品
2 省国资委课题申报
湘小协 启动申报 → 深度研究 背景调研 → MECE 框架搭建 → 产业+宏观 数据支撑 → 课题申报书 输出 → 湘小印 格式审核
3 城投退平台转型路径分析
湘小研 统筹分析 → 深度研究 转型案例 → 产业分析 机会识别 → 数据分析 量化评估 → 七步成诗 路径设计 → 湘小印 出品报告
4 季度经营分析报告
湘小协 启动季度报告 → 政策追踪 政策动态 → 宏观分析 经济形势 → 数据分析 指标更新 → 深度研究 综合研判 → 湘小印 标准化输出
